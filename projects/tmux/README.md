# tmux

tmux configuration and session management workflow.

## Structure

```
config/    # ~/.tmux.conf and related files
notes/     # Learning notes, workflow tips

```

## Setup

To use this config, symlink or copy:

```bash
ln -s $(pwd)/config/.tmux.conf ~/.tmux.conf
```
