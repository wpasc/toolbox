# NeoVim

NeoVim setup, configuration, and learning.

## Structure

```
config/    # ~/.config/nvim/ - actual dotfiles
notes/     # Learning notes, keybinding references
scripts/   # Neovim-related utilities
```

## Setup

To use this config, symlink or copy to `~/.config/nvim/`:

```bash
ln -s $(pwd)/config ~/.config/nvim
```
